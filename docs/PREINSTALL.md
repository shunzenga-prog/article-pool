# 安装前准备

## 必需目录

在安装前，请确保以下目录存在：

```bash
mkdir -p ~/.openclaw/workspace/reports/materials
mkdir -p ~/.openclaw/workspace/reports/images
mkdir -p ~/.openclaw/workspace/templates
```

## 环境变量（可选）

可以通过环境变量自定义输出路径：

```bash
# 设置输出目录
export OUTPUT_DIR=~/.openclaw/workspace/reports/materials
```
