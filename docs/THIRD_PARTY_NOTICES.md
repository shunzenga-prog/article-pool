# Third-Party Skill Notes

Article Pool includes workflow ideas adapted from external skill repositories.

## ConardLi/garden-skills

- Repository: https://github.com/ConardLi/garden-skills
- License: MIT License
- Integrated as adapted workflow guidance, not as a vendored dependency.
- Referenced concepts:
  - `gpt-image-2`: mode-aware image prompting and honest no-image fallback.
  - `kb-retriever`: layered local knowledge retrieval and progressive search.
  - `web-video-presentation`: article/script to 16:9 web-video planning.
  - `web-design-engineer`: visual quality gates and anti-generic design rules.

No executable files from this repository are vendored in Article Pool in this integration slice.
