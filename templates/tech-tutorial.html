终端


# 创建项目目录和虚拟环境
mkdir fastapi-ai-api && cd fastapi-ai-api
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 安装依赖
pip install fastapi uvicorn openai

执行完后，项目目录下会多一个 venv 文件夹，里面是独立的 Python 环境。之后安装的任何包都不会污染系统 Python。

⚠️ 常见踩坑：如果 pip install 报权限错误，检查是否激活了虚拟环境（终端前面应该有 (venv) 标识）

核心概念：异步编程（Async）

FastAPI 的异步特性（async/await）让它在调用外部 API 时不会阻塞其他请求，一个进程就能处理上千并发。

可以理解为：同步是"点完一道菜、等上菜、吃完、再点下一道"；异步是"一次性点完所有菜，哪道先上就先吃哪道"。
