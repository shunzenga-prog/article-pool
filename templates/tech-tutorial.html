用 FastAPI 搭建你的第一个 AI API 服务

2026.04.27 · 预计阅读 8 分钟

开始前的准备

▸ Python 3.10+ 已安装

▸ 基本的 Python 语法知识

▸ 一个 OpenAI API Key（或其他 LLM 服务）

1

安装依赖

首先创建一个虚拟环境并安装所需的库。FastAPI 用于构建 API，uvicorn 作为 ASGI 服务器，openai 用于调用大模型。

       终端 


# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 安装依赖
pip install fastapi uvicorn openai

2

编写核心代码

创建 main.py 文件，编写以下代码：

from fastapi import FastAPI
from pydantic import BaseModel
from openai import OpenAI

app = FastAPI()
client = OpenAI(api_key="your-api-key")

class Prompt(BaseModel):
    text: str

@app.post("/api/chat")
async def chat(prompt: Prompt):
    response = client.chat.completions.create(
        model="gpt-4o",
        messages=[{"role": "user", "content": prompt.text}]
    )
    return {"reply": response.choices[0].message.content}

关键概念

FastAPI 的异步特性（async/await）使得它在处理 I/O 密集型任务（如调用外部 API）时效率极高，一个进程就能处理数千并发请求。

3

运行并测试

启动服务，然后访问自动生成的 API 文档：

uvicorn main:app --reload

打开浏览器访问：

http://localhost:8000/docs

}

常见问题

Q: 如何部署到生产环境？

A: 推荐使用 Railway、Fly.io 或自建 Docker 容器。FastAPI 官方文档有详细的部署指南。

Q: 如何添加流式响应？

A: 使用 FastAPI 的 StreamingResponse，配合 OpenAI 的 stream=True 参数即可实现。
