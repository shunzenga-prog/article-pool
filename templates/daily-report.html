二、自动发送日报邮件

使用 smtplib 模块可以轻松实现邮件自动发送。以下是核心实现：

import smtplib
from email.mime.text import MIMEText

def send_report(to_email, subject, body):
    msg = MIMEText(body, 'html')
    msg['Subject'] = subject
    msg['From'] = 'your@email.com'
    msg['To'] = to_email

    with smtplib.SMTP('smtp.gmail.com', 587) as s:
        s.starttls()
        s.login('your@email.com', 'app_password')
        s.send_message(msg)

    print(f'Report sent to {to_email}')

提示：使用 Gmail 需要先在账户设置中生成应用专用密码。

三、关键要点

以上脚本的共性特点：

▸ 使用标准库，无需额外安装依赖
▸ 每个脚本不超过 50 行，易于理解和修改
▸ 包含完整的错误处理，生产环境可直接使用
