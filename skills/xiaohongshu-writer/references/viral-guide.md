# 小红书爆款笔记创作指南

> 来源：冲浪龙虾研究 | 2026-03-30

---

## 一、爆款标题四大公式

### 公式 1: 成果 + 时间 + 方法
```
结构：{时间周期}{量化成果}！我的{具体方法}
示例："7 天涨粉 10w+！我的 Midjourney 入门教程"
变体："3 个月 AI 副业赚 10w，我的实操路径大公开"
```
**适用场景**: 教程类、成长记录类

---

### 公式 2: 数字 + 利益 + 人群
```
结构：{数字}个{工具/方法}，{目标人群}{核心利益}
示例："8 个 AI 神器，打工人效率翻倍🔥"
变体："5 个提示词模板，小白也能写出爆款文案"
```
**适用场景**: 工具合集、资源整理

---

### 公式 3: 收入 + 可复制
```
结构：靠{方法}月入{金额}，普通人也能复制{emoji}
示例："靠 AI 副业月入 3w+，普通人也能复制💰"
变体："用 AI 做海报，每月被动收入 5000+，新手友好"
```
**适用场景**: 变现案例、副业分享

---

### 公式 4: 承诺 + 资源
```
结构：{结果承诺}！{资源类型}公开{emoji}
示例："篇篇爆款！我的提示词模板公开📝"
变体："零基础学会 AI 绘画！完整教程 + 素材包免费领"
```
**适用场景**: 教学资源、模板分享

## 二、封面设计五元素

| 元素 | 出现频率 | 作用 | 制作建议 |
|------|----------|------|----------|
| 对比图 | 80% | 视觉冲击 | 左右/上下对比，前后变化 |
| 数字突出 | 90% | 增强可信度 | 放大数字，用醒目颜色 |
| emoji 点缀 | 95% | 增加活泼感 | ✨🎨💰🔥📝🤖 |
| 大字标题 | 85% | 核心利益点 | 不超过 3 行，字号够大 |
| 标签贴纸 | 70% | 降低门槛 | "免费""新手友好""真实记录" |

### 封面类型推荐

| 内容类型 | 推荐封面 |
|----------|----------|
| 教程类 | 对比图（前后效果） |
| 工具合集 | 九宫格图标 |
| 变现案例 | 收入截图（打码）+ 人物出镜 |
| 模板类 | 代码/提示词截图 |
| 数字人 | AI 虚拟人形象 + 工作场景 |

## 三、正文风格（300-800 字）

### 结构模板

```
开头（50 字）：
- 个人经历/成果展示
- 引发共鸣

正文（200-500 字）：
用 emoji 序号（小红书特色）

例：
✅ 1️⃣ 先搞定选题（这是关键）
✅ 2️⃣ 再写大纲（别急着写正文）
✅ 3️⃣ 最后润色（加 emoji）

结尾（50 字）：
- 引导互动："评论区领取..."
- "收藏防走丢"
```

### ⚠️ 避免教科书式

| ❌ 正经式 | ✅ 小红书风格 |
|----------|--------------|
| 一、选题的重要性 | 1️⃣ 先搞定选题（这是关键） |
| 二、大纲的设计方法 | 2️⃣ 再写大纲（别急着写正文） |
| 长段落论述 | 每点不超过 3 行 |

## 四、5 个可复制模板

### 模板 1: 变现案例型（高点赞首选）
```
标题：靠 AI 副业月入 X 万，普通人也能复制💰

开头：我是一名[身份]，用 AI [时间]赚了[金额]

正文：
1️⃣ 我是怎么开始的
2️⃣ 具体怎么做（3-5 步）
3️⃣ 踩过的坑
4️⃣ 给新手的建议

结尾：想学的评论区扣"1"，我教你
```

### 模板 2: 教程入门型（高收藏首选）
```
标题：X 天学会 AI [技能]！我的[工具]入门教程📝

开头：零基础小白，X 天从 0 到[成果]

正文：
1️⃣ 工具准备
2️⃣ 核心步骤（配截图）
3️⃣ 常见问题解答
4️⃣ 进阶技巧

结尾：完整教程 PDF 评论区领取
```

### 模板 3: 工具合集型（稳定爆款）
```
标题：2026 年必藏的 X 个 AI 神器！[人群]效率翻倍🔥

开头：整理了 X 个我常用的 AI 工具，亲测好用

正文：
1️⃣ [工具名] - 一句话说明 + 使用场景
2️⃣ [工具名] - 一句话说明 + 使用场景
...

结尾：收藏防走丢，下期分享更多神器
```

### 模板 4: 模板资源型（超高收藏率）
```
标题：篇篇爆款！我的[类型]模板公开[emoji]

开头：用这些模板，我 X 个月涨粉 X 万

正文：
模板 1：[标题模板]
模板 2：[正文模板]
模板 3：[提示词模板]

结尾：想学更多的评论区告诉我
```

### 模板 5: 新奇概念型（高传播性）
```
标题：我的 AI [概念]帮我[动作]，每天[收益]🤖

开头：发现了一个新玩法，X 天试了 X 次

正文：
1️⃣ 是什么
2️⃣ 怎么做
3️⃣ 有什么坑

结尾：想了解的评论区扣"1"
```

## 六、标签策略（8-10 个）

### 标签组合公式

```
核心标签（2-3 个）+ 人群标签（2-3 个）+ 热门标签（3-4 个）

例：
核心：#AI #AI绘画 #效率工具
人群：#打工人 #自媒体人 #副业党
热门：#涨粉 #搞钱 #新手教程 #干货分享
```
