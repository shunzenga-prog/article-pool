# 小红书爆款笔记创作

专为小红书平台优化的笔记创作技能。

## 触发场景

- "写一篇小红书笔记"
- "发小红书"
- "小红书创作"
- "笔记怎么写"

## 创作流程

1. **确认选题** → 用户提供话题或从选题库选
2. **验证时效** → ⚠️ 热点类必查
3. **生成标题** → 爆款公式
4. **撰写正文** → 小红书风格
5. **封面建议** → 视觉设计
6. **标签推荐** → SEO优化

## ⚠️ 资料验证规范（核心原则）

**搜索参考资料时，必须验证来源和时间，避免使用过期消息！**

### 验证流程（搜资料时）

1. 搜索原文发布时间
2. 判断是否过时
3. 检查是否有最新进展/辟谣
4. 决定是否使用该资料

### 时效判断标准（筛选资料用）

| 发布时间 | 使用建议 |
|----------|----------|
| < 24h | 可放心使用 |
| 24-48h | 需补充最新进展 |
| > 48h | 转分析角度 |
| 已辟谣 | 不使用 |

### ⚠️ 注意

**创作内容不需要强制标注来源时间**，验证是用于筛选资料。

## 爆款标题公式

详见 `references/viral-guide.md`

### 四大公式

| 公式 | 结构 | 适用场景 |
|------|------|----------|
| 成果+时间+方法 | "7天涨粉10w+！我的教程" | 教程类 |
| 数字+利益+人群 | "8个AI神器，打工人效率翻倍🔥" | 工具合集 |
| 收入+可复制 | "靠AI副业月入3w+，普通人也能复制💰" | 变现案例 |
| 承诺+资源 | "篇篇爆款！我的模板公开📝" | 模板分享 |

## 正文结构（300-800 字）

```
开头（50 字）：
- 个人经历/成果展示
- 引发共鸣

正文（200-500 字）：
用 emoji 序号（小红书特色）

例：
✅ 1️⃣ 先搞定选题（这是关键）
✅ 2️⃣ 再写大纲（别急着写正文）
❌ 一、选题的重要性
❌ 二、大纲的设计方法

要点说明：
- 用 emoji 序号（小红书特色）
- 每点不超过 3 行
- 加 emoji 增活泼感

结尾（50 字）：
- 引导互动
- "评论区领取..."
- "收藏防走丢"
```

### ⚠️ 写作风格要求

**小红书特点：短平快，用 emoji 序号没问题！**

但还是要避免教科书式表达：
- ❌ "第一、第二、第三" → ✅ 用 emoji 序号 + 口语化说明
- ❌ 长段落论述 → ✅ 每点不超过 3 行
- ❌ 正经标题 → ✅ 加 emoji，口语化

## 封面设计建议

| 元素 | 频率 | 建议 |
|------|------|------|
| 对比图 | 80% | 前后效果对比 |
| 数字突出 | 90% | 放大数字，醒目颜色 |
| emoji 点缀 | 95% | ✨🎨💰🔥📝🤖 |
| 大字标题 | 85% | 不超过 3 行 |
| 标签贴纸 | 70% | "免费""新手友好" |

**封面类型：**
- 教程类 → 对比图（前后效果）
- 工具合集 → 九宫格图标
- 变现案例 → 收入截图（打码）+ 人物出镜
- 模板类 → 代码/提示词截图
