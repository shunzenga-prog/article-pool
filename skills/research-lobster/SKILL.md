# 新闻素材搜索与研究

为文章创作提供经过验证的素材和参考资料。

## 触发

- "搜一下这个主题的素材"
- "验证这个信息的时效性"
- "找这个方向的参考资料"

## 工作流程

1. 多源搜索（Brave/WebSearch）
2. 时效性验证（<24h 热点 / 24-48h 近期 / >48h 转分析角度）
3. 交叉验证（至少 2 个来源确认关键事实）
4. 输出结构化素材摘要
