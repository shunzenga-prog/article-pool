# 文章终审检查清单

**排版完成后，必须进行全文审阅！**

---

## 一、AI 味道检查（最关键）

### 扫描禁用词汇

**AI 词汇耻辱榜**（出现≥3个就要改）：

| 类别 | 禁用词 |
|------|--------|
| 经典款 | delve, tapestry, intricate, meticulous, pivotal, vibrant, testament, interplay, garner, landscape, Additionally, valuable, boasts, bolstered, enduring, crucial |
| 仍在滥用 | align with, fostering, enhance, showcasing, highlighting, underscore, emphasizing |
| 中文对应 | 深入探讨、交织图景、错综复杂、一丝不苟、关键的、充满活力的、证明、相互作用、获得、格局、此外、有价值的、拥有、增强、持久的、至关重要的、保持一致、培育、展示、突出、强调 |

### 检查结构陷阱

- [ ] **意义膨胀** — "标志着...关键时刻"、"代表了...重大转变" → 删除或改写
- [ ] **"-ing"尾巴** — "收入增长40%，突显了..." → 拆成两句
- [ ] **"尽管面临挑战"公式** → 删除公式化包装
- [ ] **"不仅X，而且Y"** → 直接表达观点
- [ ] **三段式列表** — "创新的、变革性的、突破性的" → 精简
- [ ] **同义词循环** — "这位艺术家...这位创意远见者..." → 直接重复名字
- [ ] **避免"是"和"有"** — "作为展览空间" → 改成"是展览空间"

### AI 味道自检

| 检查项 | AI 味 | 人味 |
|--------|-------|------|
| 动词 | 花哨（作为、拥有、展示） | 简单（是、有、做了） |
| 语气 | 自我吹捧（这很重要因为...） | 陈述事实 |
| 句长 | 长句、多重从句 | 短句也可以 |
| 归因 | 模糊（专家认为...） | 具体（XXX写道...） |
| 标题 | Title Case | 小写也可以 |
| 加粗 | 过度 | 稀疏 |
| 缩写 | 避免 | 用缩写（它's、don't） |

## 二、写作技巧验证

### 三层论述法

每个核心观点是否包含：

| 层次 | 内容 | 检查 |
|------|------|------|
| 第一层 | 是什么（事实） | [ ] |
| 第二层 | 为什么（原因） | [ ] |
| 第三层 | 怎么做（行动） | [ ] |

**示例**：
- ❌ 只说"Anthropic封杀OpenClaw"（一层）
- ✅ "Anthropic封杀OpenClaw（是什么），原因是商业利益（为什么），用户可以转向国产替代（怎么做）"

### 5W1H 检查

关键信息是否完整：

- [ ] Who — 谁做的？
- [ ] What — 发生了什么？
- [ ] When — 什么时候？
- [ ] Where — 在哪里？
- [ ] Why — 为什么？
- [ ] How — 怎么做的？

### SCQA 结构检查

- [ ] **S（情境）** — 是否用熟悉的事实引入？
- [ ] **C（冲突）** — 是否制造了矛盾/问题？
- [ ] **Q（疑问）** — 是否提出了读者关心的问题？
- [ ] **A（答案）** — 是否给出了清晰的解决方案？

## 三、内容价值检查

### 信息密度

- [ ] 每屏（约300字）是否有至少1个核心观点？
- [ ] 是否有具体数据/案例支撑？
- [ ] 是否有可操作的建议？

### 独特性

- [ ] 是否有独特的观点/角度？
- [ ] 是否有原创的分析？
- [ ] 是否有个人风格的表达？

### 实用性

- [ ] 读者看完能学到什么？
- [ ] 是否有"干货"？
- [ ] 是否有"金句"可转发？

## 四、真人感检查（8项≥6项通过）

| 检查项 | 通过标准 |
|--------|----------|
| 1. 个人经历/感受 | 有第一人称表达（我认为、我的判断是） |
| 2. 具体细节 | 有时间、地点、数字等具体信息 |
| 3. 情绪波动 | 有惊讶、兴奋、担忧等真实情绪 |
| 4. 不完美表达 | 有口语化、有停顿、有转折 |
| 5. 对话感 | 像在跟读者聊天，不是讲课 |
| 6. 独特观点 | 有自己的判断，不是人云亦云 |
| 7. 场景描写 | 有具体的场景、画面感 |
| 8. 行动号召 | 结尾有明确的引导 |

**⚠️ 禁止编造虚假体验！**
- ❌ 不能写"我试过了"、"我亲自体验过"
- ✅ 真实体验必须真的发生过
- ✅ 观点可以有自己的（"我认为"、"我的判断是"）

## 五、样式观感检查

### 排版规范

- [ ] 正文字号 15-16px？
- [ ] 行距 1.75-2.0？
- [ ] 段落≤5 行？
- [ ] 表格≤2 个？
- [ ] 颜色≤4 种？
- [ ] 图片有说明？
- [ ] 重点有标注？

### 标题层级

- [ ] H2 标题是否清晰区分章节？
- [ ] H3 标题是否合理划分小节？
- [ ] 标题是否有吸引力？

### 阅读节奏

- [ ] 是否有长段落需要拆分？
- [ ] 是否有信息堆砌需要精简？
- [ ] 是否有视觉疲劳点需要插入图片？

## 八、最终清单

### 发布前必查

- [ ] **AI 味道** — 扫描禁用词汇、结构陷阱
- [ ] **写作技巧** — 三层论述、5W1H、SCQA
- [ ] **内容价值** — 信息密度、独特性、实用性
- [ ] **真人感** — 8 项≥6 项通过
- [ ] **样式观感** — 排版规范、标题层级、阅读节奏
- [ ] **时效准确** — 来源可靠、时间新鲜
- [ ] **平台适配** — 符合平台规范

### 审阅方式

1. **通读全文** — 像读者一样阅读，不顺的地方标记
2. **出声朗读** — 发现拗口的句子
3. **手机预览** — 检查排版效果
4. **他人试读** — 收集真实反馈
