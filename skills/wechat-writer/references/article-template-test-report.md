### ✅ 2. 配色方案：蓝紫渐变科技感

**主色调**:
- 正文颜色：`#333333`（深灰色，符合需求）
- 引用背景：`#f8f9fa`（浅灰色，符合需求）
- 品牌渐变色：`linear-gradient(135deg, #667eea 0%, #764ba2 100%)`（蓝紫渐变）

**强调色**:
- 高亮：`#fff3cd`（暖黄色）
- 警告：`#f8d7da` + `#721c24`（红色系）
- 成功：`#d4edda` + `#155724`（绿色系）

**暗黑模式**:
- 背景：`#1a1a1a`（深灰色，非纯黑）
- 文字：`#e0e0e0`（浅灰色，非纯白）
- 自动适配系统主题

```css

```

- [x] 代码块横向滚动支持

⚠️ **表格数量** - 模板支持表格，但建议 ≤2 个/篇（内容规范，非技术限制）

## 使用指南

### 方法一：直接复制 HTML

1. 打开 `article-template.html`
2. 复制全部代码
3. 粘贴到微信公众号编辑器
4. 替换示例内容

### 方法二：Markdown 转换

```bash
# 使用 pandoc 转换
pandoc article.md -o article.html --template=skills/wechat-writer/references/article-template.html
```

### 方法三：脚本发布

```bash
# 使用公众号发布脚本
python3 scripts/wechat_publish.py <文章路径> [标题]
```
