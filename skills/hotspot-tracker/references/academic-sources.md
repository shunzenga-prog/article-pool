## 如何判断论文价值

| 指标 | 标准 |
|------|------|
| 引用量 | >100 次（半年内）= 热门 |
| GitHub Stars | >1000 = 实用性强 |
| 媒体报道 | 有主流媒体报道 = 影响力大 |
| 作者机构 | 顶级实验室 = 可信度高 |

## 学术内容创作角度

1. **论文解读** - 通俗解释论文核心贡献
2. **技术对比** - 与已有方法对比优劣
3. **实操指南** - 如何复现/使用论文方法
4. **行业影响** - 论文对行业的影响分析

## ⚠️ 学术诚信

- 准确引用论文标题、作者、机构
- 不曲解论文结论
- 标注"论文解读"而非"新闻"
